----- Start VPNGate -----
>>> Start your devices keep connected VPN
$ bash /home/user/../auto-ovpn@yahoo.com/sh/start-vpngate.sh KR

----- Change VPNGate -----
>>> Change VPN, try to connect others VPN
$ bash /home/user/../auto-ovpn@yahoo.com/sh/edit-vpngate.sh

----- Delete VPNGate -----
>>> Clear all of VPN settings in Network Manager
$ bash /home/user/../auto-ovpn@yahoo.com/sh/del-vpngate.sh

----- Delete All VPNGate -----
>>> Clear all of VPN settings in Network Manager & OVPN files
$ bash /home/user/../auto-ovpn@yahoo.com/sh/del-all-vpngate.sh

----- Stop VPNGate -----
>>> Stop your devices keep connected VPN
$ bash /home/user/../auto-ovpn@yahoo.com/sh/stop-vpngate.sh

*** These bash program can work in GNOME/KDE ***
